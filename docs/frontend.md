# Frontend — TURBO TAXI UI

## Стек

- **Next.js 16** — React-фреймворк
- **React 19** — UI-библиотека
- **TypeScript 5** — типизация
- **Tailwind CSS 4** — стили
- **Leaflet + react-leaflet** — интерактивная карта
- **shadcn/ui (Radix)** — UI-компоненты
- **Framer Motion** — анимации
- **Lucide React** — иконки


## Установка и запуск

### 1. Перейти в папку фронтенда

```bash
cd frontend
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: **http://localhost:3000**

### 4. Сборка для продакшена

```bash
npm run build
npm start
```

## Доступные скрипты

| Команда         | Описание                          |
|-----------------|-----------------------------------|
| `npm run dev`   | Запуск dev-сервера (порт 3000)    |
| `npm run build` | Сборка продакшен-версии           |
| `npm start`     | Запуск собранного приложения       |
| `npm run lint`  | Проверка кода (ESLint)            |

## Требования

- **Node.js 18+**
- **npm** (идёт вместе с Node.js)

## Переменные окружения

Фронтенд не требует файла `.env` для базовой работы. Внешние сервисы подключены напрямую:

| Сервис               | URL                                          | Назначение             |
|----------------------|----------------------------------------------|------------------------|
| OSRM                 | `https://router.project-osrm.org/route/v1/`  | Расчёт маршрутов       |
| OpenStreetMap Tiles  | `https://tile.openstreetmap.org/`             | Отображение карты      |
| Backend API          | `http://localhost:8000`                       | Расчёт стоимости       |

## Функциональность

- Интерактивная карта с центром на Худжанде (40.2833°N, 69.6222°E)
- Выбор точек посадки и высадки кликом по карте
- Автоматический расчёт маршрута через OSRM
- Отображение расстояния и времени в пути
- Выбор тарифа (Эконом / Комфорт / Бизнес)
- Динамический расчёт стоимости поездки
- Адаптивный интерфейс (мобильный + десктоп)
