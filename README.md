#Taxi Platform

Taxi platform prototype with:

- FastAPI backend
- Next.js frontend
- OpenStreetMap routing
- Automatic fare calculation
- Tariff selection

## Features

- Select pickup and destination on map
- Automatic route calculation
- Distance and duration detection
- Dynamic taxi pricing
- Modern taxi-style UI

## Tech Stack

Backend:
- Python
- FastAPI

Frontend:
- Next.js
- React
- Leaflet map

Routing:
- OpenStreetMap
- OSRM
